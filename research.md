## 1. Топ-5 конвертеров: что делает их хорошими

### 1.1 Google Unit Converter (встроен в поиск)

**Сильные стороны:**
- Мгновенный доступ -- вбил "15 kg to lbs" и готово
- Real-time обновление при вводе (без кнопки "Convert")
- Минимум UI: два поля, два dropdown, кнопка swap
- Автоопределение категории по единицам
- Формулы конвертации прозрачны

**Что взять:**
- Двустороннее поле ввода с мгновенным пересчётом
- Swap-кнопка между полями
- Минимализм интерфейса -- ничего лишнего

### 1.2 iOS Calculator / Apple Unit Converter

**Сильные стороны:**
- Нативный iOS look: большие цифры, тёмная тема, минимум хрома
- Категории через горизонтальные tabs/pills наверху
- Плавные анимации при переключении
- Haptic feedback (native)
- Крупный шрифт результата -- главное число бросается в глаза

**Что взять:**
- Dark theme с iOS-стилем pills/tabs для категорий
- Крупный шрифт для результата (44-48px)
- Rounded corners, blur-эффекты (backdrop-filter)
- Минимальная цветовая палитра: белый текст на тёмном фоне, accent color

### 1.3 XE.com (валюты)

**Сильные стороны:**
- Доверенный источник курсов валют
- Показывает время последнего обновления курса
- Графики исторических курсов
- Quick-pick популярных пар (USD/EUR, EUR/GBP)
- Чёткое разделение: "Amount", "From", "To"

**Что взять:**
- Показывать timestamp курсов (для валют с оговоркой "статичный курс")
- Quick-pick популярных пар
- Три чётких элемента: количество, откуда, куда

### 1.4 UnitConverters.net

**Сильные стороны:**
- Огромное количество единиц (включая редкие)
- Таблица "common conversions" рядом с конвертером
- Формула пересчёта показана под результатом
- SEO-оптимизация (каждая пара = отдельная страница)

**Что взять:**
- Показ формулы под результатом ("1 km = 0.621371 mi")
- Предустановленные популярные конверсии как quick links

### 1.5 ConvertPad (мобильное приложение)

**Сильные стороны:**
- Кастомные единицы: можно добавить свои
- Конвертация в реальном времени для ВСЕХ единиц категории одновременно
- Избранные пары -- быстрый доступ
- Клавиатура встроена в интерфейс (не системная)

**Что взять:**
- Показ нескольких результатов сразу (ввёл 100 кг -- видишь фунты, унции, граммы одновременно)
- Избранные пары для повторного использования

## 2. Категории единиц (MVP)

### 2.1 Длина

| Единица | Код | Коэффициент (к метру) |
|---------|-----|----------------------|
| Миллиметр | mm | 0.001 |
| Сантиметр | cm | 0.01 |
| Метр | m | 1 |
| Километр | km | 1000 |
| Дюйм | in | 0.0254 |
| Фут | ft | 0.3048 |
| Ярд | yd | 0.9144 |
| Миля | mi | 1609.344 |

### 2.2 Вес (масса)

| Единица | Код | Коэффициент (к грамму) |
|---------|-----|----------------------|
| Миллиграмм | mg | 0.001 |
| Грамм | g | 1 |
| Килограмм | kg | 1000 |
| Фунт | lb | 453.592 |
| Унция | oz | 28.3495 |

### 2.3 Температура

Тут нельзя через коэффициент -- нужны формулы:

```
C -> F: F = C * 9/5 + 32
F -> C: C = (F - 32) * 5/9
C -> K: K = C + 273.15
K -> C: C = K - 273.15
F -> K: K = (F - 32) * 5/9 + 273.15
K -> F: F = (K - 273.15) * 9/5 + 32
```

### 2.4 Валюты (статичные курсы)

| Валюта | Код | Курс к USD (примерный) |
|--------|-----|----------------------|
| Доллар США | USD | 1 |
| Евро | EUR | 0.92 |
| Российский рубль | RUB | 87.5 |
| Казахстанский тенге | KZT | 510 |
| Фунт стерлингов | GBP | 0.79 |

**Важно:** курсы статичные, без API. Обязательно показывать disclaimer: "Курсы приблизительные, обновлены DD.MM.YYYY".

## 3. UX-паттерны: анализ и рекомендации

### 3.1 Выбор единиц

| Паттерн | Плюсы | Минусы | Вердикт |
|---------|-------|--------|---------|
| Dropdown (select) | Привычно, компактно | На мобиле native picker плохо кастомизируется | Базовый вариант |
| Tabs/Pills | Визуально чисто, один тап | Не скалируется при 8+ единиц | Для категорий -- да, для единиц -- нет |
| Search/Autocomplete | Быстро при большом списке | Overkill для 5-8 единиц | Для MVP не нужно |
| Custom dropdown | Полный контроль стиля | Сложнее в реализации | Оптимальный для dark iOS theme |

**Рекомендация:** Tabs/pills для категорий (длина/вес/температура/валюта). Custom styled dropdown для выбора единиц внутри категории.

### 3.2 Двунаправленный ввод

Это must-have. Паттерн:
- Два input-поля ("from" и "to")
- Ввод в ЛЮБОЕ из полей пересчитывает другое
- Фокус определяет направление
- Нет кнопки "Convert" -- пересчёт на каждый keyup (с debounce 100ms)

Техническая реализация (single HTML, vanilla JS):
```
// Определяем активное поле, пересчитываем пассивное
inputFrom.addEventListener('input', () => convert('from'));
inputTo.addEventListener('input', () => convert('to'));
```

### 3.3 Swap-кнопка

Обязательный элемент. Размещение: между двумя полями, иконка стрелок (двунаправленная). При нажатии:
1. Поменять выбранные единицы местами
2. Пересчитать значение
3. Лёгкая анимация поворота иконки (CSS transition 0.3s)

### 3.4 Популярные пары (Quick picks)

Под конвертером -- небольшие chips/pills:
- Длина: km <-> mi, cm <-> in, m <-> ft
- Вес: kg <-> lb, g <-> oz
- Температура: C <-> F
- Валюты: USD <-> RUB, USD <-> KZT, EUR <-> USD

При клике: устанавливает пару и фокусирует поле ввода.

### 3.5 Real-time обновление

- `input` event (не `change`, не `keyup`) -- работает с любым вводом включая paste
- Debounce не нужен для простых математических операций -- расчёт мгновенный
- Форматирование числа: разделители тысяч, максимум 6 знаков после запятой (trim trailing zeros)

## 6. Технические заметки для реализации

### Архитектура данных (JS)

```javascript
const units = {
  length: {
    name: 'Length',
    base: 'm',
    units: {
      mm:  { name: 'Millimeter', factor: 0.001 },
      cm:  { name: 'Centimeter', factor: 0.01 },
      m:   { name: 'Meter',      factor: 1 },
      km:  { name: 'Kilometer',  factor: 1000 },
      in:  { name: 'Inch',       factor: 0.0254 },
      ft:  { name: 'Foot',       factor: 0.3048 },
      yd:  { name: 'Yard',       factor: 0.9144 },
      mi:  { name: 'Mile',       factor: 1609.344 },
    }
  },
  // weight, currency -- аналогично через factor
  // temperature -- через отдельные функции convert(value, from, to)
};
```

### Формула конвертации (factor-based)
```javascript
function convert(value, fromFactor, toFactor) {
  return value * fromFactor / toFactor;
}
// 100 km -> mi: 100 * 1000 / 1609.344 = 62.137
```

### Температура (отдельная логика)
```javascript
const tempConvert = {
  'C-F': v => v * 9/5 + 32,
  'F-C': v => (v - 32) * 5/9,
  'C-K': v => v + 273.15,
  'K-C': v => v - 273.15,
  'F-K': v => (v - 32) * 5/9 + 273.15,
  'K-F': v => (v - 273.15) * 9/5 + 32,
};
```

### Размер файла

Оценка для single HTML: ~15-20 KB (без минификации). С inline CSS и JS.

## 7. Вердикт

Конвертер единиц -- идеальный проект для single HTML file. Математика тривиальная, UI компактный, зависимости нулевые. Ключевой фактор успеха -- не функционал, а polish: iOS-стиль, плавные анимации, мгновенный отклик. Пользователь должен чувствовать что это нативное приложение, а не веб-страница.

Главные риски:
- **Валюты без API** -- курсы устареют. Решение: жирный disclaimer + дата обновления + P2 план на API
- **Custom dropdown на мобиле** -- нативный `<select>` выглядит не-iOS на Android. Решение: custom dropdown с touch-событиями
- **Точность float** -- `0.1 + 0.2 != 0.3`. Решение: округление до разумного количества знаков (6 max)

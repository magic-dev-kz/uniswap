## Уникальное преимущество

Режим "все единицы" -- вводишь число один раз и видишь результат во ВСЕХ единицах категории одновременно. Конкуренты заставляют выбирать пару "из A в B", мы показываем всё сразу. Плюс real-time расчёт без кнопки "Конвертировать" -- результат появляется на каждый нажатый символ. Zero-tap experience.

## Рекомендации по скриншотам

**Скриншот 1**: Режим "все единицы" -- введено `100` в поле "кг", ниже карточки со всеми весовыми единицами (г, фунты, унции). Подпись: "Все единицы. Одним взглядом." Тёмный фон, оранжевый акцент категории "Вес".

**Скриншот 2**: Конвертация температуры с анимацией swap (стрелки между C и F). Подпись: "Мгновенный результат при вводе." Красный акцент температуры -- яркий, привлекает внимание в поиске.

**Скриншот 3**: Момент копирования -- карточка с результатом подсвечена, галочка "Скопировано". Подпись: "Скопируй одним тапом." Демонстрирует ключевой flow: ввёл -> увидел -> скопировал.

### 1.2 Разбор каждого

**Apple Calculator (iOS 18+)**
- Главная угроза. Уже на каждом iPhone, не надо ничего ставить.
- 15 категорий включая angle, area, data, energy, force, fuel, pressure, speed.
- Валюты: ~105 валют, курсы от Yahoo Finance, показывает дату обновления.
- Math Notes: можно написать "10 kg to pounds" и получить ответ.
- Слабости: нет режима "все единицы сразу", нет favorites, нет копирования тапом, интерфейс утилитарный.

**Google Search Converter**
- Вбил "15 kg to lbs" -- ответ без открытия приложения.
- Автоопределение категории по контексту запроса.
- Dropdown с unit picker, realtime при вводе.
- Слабости: требует интернет, нет PWA/offline, нет "all units", нельзя установить.

**Units - Pro Unit Converter (iOS)**
- 800+ единиц, 43 категории. Бесплатно, без рекламы.
- 88 валют с ежедневным обновлением.
- Favorites, поиск единиц, Apple Watch.
- Слабости: интерфейс перегружен для базовых задач, много экранов навигации.

**Converter+**
- Швейцарский нож: конвертер + калькуляторы (mortgage, tip, fuel).
- Custom templates -- можно создавать свои экраны конвертации.
- User-contributed templates: скачиваешь чужие шаблоны.
- 167 валют, обновление каждые 6 часов.
- Слабости: интерфейс устаревший, перегружен функциями. "Feels like 2018 app."

**GlobeConvert**
- 350+ единиц, 160+ валют.
- Favorites/presets -- ключевая фича, пин на часто используемые конвертации.
- Offline: кэширует последние курсы валют.
- Widget поддержка.
- Слабости: freemium, реклама в бесплатной версии.

**Amount Plus**
- 700+ единиц, 30+ категорий.
- Custom units: можно добавить свои единицы измерения.
- Shortcuts для quick access с Home Screen.
- Apple Watch.

## 2. Что делает конвертер "would I pay $2.99?"

Проанализировав платные конвертеры, вот что отличает "готов заплатить" от "бесплатного хватит":

### 2.1 Таблица фич по ценовой готовности

| Фича | Бесплатные имеют | Платные имеют | UniSwap v1 |
|------|-----------------|---------------|------------|
| Realtime конвертация | Да | Да | **Да** |
| Swap direction | Да | Да | **Да** |
| "All units" view | Редко | Да | **Да** |
| Copy to clipboard | Иногда | Всегда | **Да** |
| Favorites / Pinned pairs | Нет | Да | Нет |
| Currency conversion (live) | Нет | Да | Нет |
| Custom units | Нет | Да | Нет |
| Calculator внутри | Нет | Иногда | Нет |
| History | Нет | Да | Нет |
| Widgets / Shortcuts | Нет | Да | Нет |
| Offline | Иногда | Всегда | **Да** |
| Dynamic accent per category | Нет | Нет | **Да** |
| Overlay unit selector | Нет | Нет | **Да** |

### 2.2 Ключевые выводы

**"$2.99 порог" пробивают три фичи:**

1. **Favorites / Pinned conversions** -- возвращаемость. Пользователь не гуглит каждый раз, а открывает приложение, потому что его пара уже сохранена. Это превращает утилиту в привычку.

2. **Currency с live rates** -- единственная категория, где данные меняются. Это создаёт причину открывать приложение повторно. Но требует API и онлайн.

3. **History** -- "что я конвертировал вчера?". Маленькая фича, но сильно снижает трение при повторных задачах.

**Калькулятор внутри и custom units -- переоценены.** Большинство пользователей их не используют. Это фичи для power users (<5%).

## 3. Уникальность UniSwap

### 3.1 Что уже уникально

| Наша фича | Есть у конкурентов? | Оценка уникальности |
|-----------|-------------------|-------------------|
| Dynamic accent color per category | Нет ни у кого | **Высокая** -- визуально мгновенно понятно где ты |
| Overlay unit selector (full-screen) | Нет (все используют dropdown или picker) | **Высокая** -- лучше для мобиле, чем native select |
| Single-file PWA (0 dependencies) | Крайне редко | **Высокая** -- мгновенная загрузка, 0 зависимостей |
| Полностью offline из коробки | Частично (Units Pro, GlobeConvert) | **Средняя** -- PWA offline редкость для конвертеров |
| "All units" view по категории | Units Pro, ConvertPad | **Средняя** -- не уникально, но реализация чистая |

### 3.2 Что можно добавить для уникальности

**"Conversion context" -- формула + пояснение.**
Не просто "100 kg = 220.462 lb", а под результатом: "1 kg = 2.205 lb". Большинство конвертеров не показывают формулу. Это обучающий элемент -- человек запоминает коэффициент. Уже частично есть в research.md как рекомендация.

**"Quick math in input"** -- ввести "2*5" в поле и получить конвертацию 10.
Ни один конкурент (кроме Apple Math Notes) этого не делает. Но Apple Math Notes -- это отдельный режим, а у нас прямо в поле ввода.

**"Shake to swap"** -- Device Motion API. Потряс телефон -- поменял направление. Гимик, но запоминается.

## 4. TOP-3 фичи для v2 (от 9.2 к 9.5+)

### Приоритет 1: Дополнительные категории

**Что:** добавить Area, Speed, Time, Data (из spec P1).

**Почему:** Apple Calculator имеет 15 категорий. Наши 4 -- слишком мало для серьёзного использования. Каждая новая категория = новый accent color = усиление нашей уникальной фичи. 8 категорий -- сладкое место (достаточно, но не перегружено).

**Влияние на оценку:** +0.1-0.15. Функциональная полнота -- базовое ожидание.

**Сложность:** Низкая. Та же архитектура (factor-based), только добавить данные.

### Приоритет 2: Favorites / Pinned pairs

**Что:** длинный тап или иконка звезды на паре -> сохраняется в localStorage -> показывается отдельным блоком наверху.

**Почему:** это фича номер один, которая отличает "открою один раз" от "это моё приложение". GlobeConvert, Units Pro, Amount Plus -- все платные конкуренты имеют favorites. У нас нет. Это самый большой gap.

**Влияние на оценку:** +0.15-0.2. Превращает утилиту в привычку.

**Сложность:** Средняя. localStorage, UI для pinned pairs, возможно drag-to-reorder.

### Приоритет 3: History (последние 10 конвертаций)

**Что:** автоматически запоминать последние конвертации. Показывать как сворачиваемый блок внизу. Тап повторяет конвертацию.

**Почему:** снижает трение при повторных задачах. Не надо вспоминать "а что я конвертировал вчера". Вместе с Favorites создаёт полный цикл "новая задача -> повтор -> закрепление".

**Влияние на оценку:** +0.1. Мелочь, но создаёт ощущение "живого" приложения.

**Сложность:** Низкая. localStorage, рендер списка, очистка старых записей.

## 5. Стратегическое позиционирование

### Где мы выигрываем уже сейчас

```
UniSwap vs Apple Calculator:  Лучше UX, "all units" view, dynamic color, overlay selector
UniSwap vs Google:             Offline, installable PWA, лучше мобильный UX
UniSwap vs Units Pro:          Чище дизайн, меньше clutter, мгновенная загрузка
UniSwap vs Converter+:         Современный дизайн, не перегружен, бесплатно без рекламы
UniSwap vs GlobeConvert:       Нет рекламы, чистый PWA, лучше дизайн
```

### Где мы проигрываем

```
vs Apple Calculator:   Меньше категорий (4 vs 15), нет валют с live rates
vs Units Pro:          Нет favorites, нет валют, меньше единиц
vs Converter+:         Нет калькуляторов, нет custom templates
vs GlobeConvert:       Нет favorites, нет widgets
```

### Ниша UniSwap

**"The fastest, cleanest unit converter that works everywhere."**

Мы не конкурируем с Units Pro по количеству единиц (800 vs наши 20+). Мы не конкурируем с Converter+ по функциональности (калькуляторы, templates). Мы конкурируем по **скорости, чистоте и удовольствию от использования**.

Наш пользователь: человек, который хочет быстро перевести kg в lb, не хочет ставить приложение из App Store, не хочет видеть рекламу, ценит красивый интерфейс.

## 7. Вердикт

UniSwap уже превосходит большинство бесплатных конвертеров по UX-качеству. Dynamic accent color и overlay selector -- реально уникальные фичи, которых нет ни у одного конкурента. Но для 9.5 нужны две вещи: **больше категорий** (закрыть gap с Apple Calculator) и **favorites** (закрыть gap с платными приложениями). History -- бонус, который закрепляет результат.

Не стоит гнаться за количеством единиц (800 как у Units Pro) или функциональностью (калькуляторы как у Converter+). Наша сила в другом: мгновенность, чистота, zero-install. Довести до 9.5 можно малой кровью.
